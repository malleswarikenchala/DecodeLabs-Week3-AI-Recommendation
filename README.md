
# AI Recommendation Logic

## Objective

Build a simple recommendation system that suggests items based on user preferences.

## Technologies Used

* Python

## Features

* Accepts user preferences
* Matches preferences with available categories
* Recommends relevant movies

## Categories

* Action
* Comedy
* Romance
* Horror

## How to Run

python3 recommendation.py

## Sample Output

Movie Recommendation System

Enter your favorite genre: action

Recommended Movies:

* John Wick
* Avengers
* Mad Max

## Conclusion

This project demonstrates the basic concept of recommendation systems using logic-based preference matching.
